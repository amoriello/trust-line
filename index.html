Opensource Password Helper Device

How does it work?

The Token looks like a USB keyboard to your computer. It can send keystrokes like a human will do on its own keyboard.

How to use this?

When you want to login somewhere, instead of typing a password:

1. Plug the Token to the computer

2. Run the iPhone App and select the account's service you want to log-in

3. The iPhone App sends the associated encrypted password to the Token, that can decrypt it (because it knows the secret key) and the Token "types" the password where your focus is.


The Token, a Bluetooth 4.0 enabled device powered by an atmega32u4 (arduino), (like the Blend Micro)
A bluetooth 4.0 compatible phone (like iPhone ≥ 4s)

Features

What about security?

The communication protocol (over BLE) uses a HMAC-SHA256 based challenge/response to authenticate.

Passwords are encrypted using AES-128 on the phone and the secret 128 bit AES key is stored inside token eeprom memory.

What if I lose my token?

It's OK, you can create a new one, and recover your passwords with the secret QR code obtained during the first pairing.

The QR code is a critical piece of information, so keep it safe.

What if I lose my phone?

It's OK, your encrypted passwords can be safely stored on any Cloud, synchronized across your devices, or simply backuped by mail. They are encrypted, and only the token has the key to decrypt them.
